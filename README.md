# 旅游景点推荐数据分析可视化系统 python1240

## 项目概述

本项目是基于Python语言的旅游景点推荐数据分析可视化系统，通过大数据技术对旅游景点进行智能推荐，帮助用户发现和选择理想的旅游目的地。

## 技术栈

### 后端
- **Python框架**：Django
- **Python版本**：3.7/3.8
- **数据库**：MySQL 5.7 或更高版本（兼容5.X版本）

### 数据库工具
- **推荐工具**：Navicat 11

### 开发工具
- **IDE**：PyCharm 或 VS Code

### 前端
- **框架**：Vue.js

## 功能模块

- 旅游景点数据采集
- 数据预处理
- 数据存储
- 数据分析与推荐算法
- 可视化展示

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 操作系统：不限
- Python环境：3.7/3.8
- 数据库：MySQL 5.7 或更高版本
- Web服务器：与Django兼容的Web服务器

## 部署步骤

1. 安装Python环境（3.7/3.8）
2. 安装MySQL数据库，并创建对应的数据表
3. 安装Django框架
4. 安装Vue.js及前端依赖
5. 部署后端服务
6. 部署前端服务
